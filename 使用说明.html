主页编辑器 · 使用说明

一个单文件网页编辑器，用来制作和修改个人 / 公司主页卡片，可导出网页或高清图片，用于朋友圈、小红书、微信群、官网等。

一、怎么打开

1. 把 index.html 下载到电脑。

2. 双击它，会用浏览器打开（推荐 Chrome 或 Edge）。

3. 打开后直接编辑，不用装软件，也不用懂代码。

💡 编辑和「导出网页」完全离线可用；只有「导出高清图片」需要联网。

二、基本操作

整个页面由一块块「积木」拼成（头部、数据、客户墙、合作步骤、联系卡片等）。

改文字

鼠标点任意文字，直接打字修改。

换图片

鼠标移到图片上，出现工具条：

  • ➖ 缩小 ／ ➕ 放大
  • ⬭ 圆形 / 方角切换
  • 📷 换图片（上传你电脑里的图）

调整积木块

鼠标移到某块上，右上角出现：

  • ⬆ 上移 ／ ⬇ 下移
  • ⧉ 复制 ／ ✕ 删除

加新积木

点顶部或底部 ＋ 添加积木，可选：圆形 Logo、图片、标题、文字段落、标签组、数据格、客户墙、步骤流程、联系卡片、分隔线、页脚等。

三、联系卡片 · 加二维码

在「联系我们」卡片里，右侧有三个快捷按钮：

  • ＋ 微信二维码 加一个二维码位（默认说明「微信扫码」）
  • ＋ 飞书二维码 加一个二维码位（默认说明「飞书扫码」）
  • ＋ 其他图片 加任意图片（如公众号码、小程序码）

每个二维码：点 📷 上传真实二维码图、用 ➖➕ 调大小、下方说明文字可点改、右上角 ✕ 删除。多个二维码自动并排。

四、改颜色和字体

点顶部 🎨 颜色 / 字体，可调：

  • 主题色：8 个预设 + 自定义取色器（改一个色，整站配色自动联动）
  • 背景调：冷灰 / 米白 / 纯白 / 浅绿 / 深色
  • 字体：黑体 / 宋体 / 楷体 / 圆体 / 等宽

五、保存成果（重要）

⚠️ 修改是在打开的网页里实时进行的，刷新或关闭页面会丢失。改完一定要导出保存！

⬇ 导出网页 下载一个干净的 .html 文件（不带编辑按钮）。可直接发给别人看，或以后用「📂 导入已有页面」重新载入接着改。

🖼️ 导出高清图片 下载一张 3 倍分辨率 PNG，适合发朋友圈、小红书、群、打印。（需联网）

六、接着上次的继续改

1. 打开本编辑器

2. 点顶部 📂 导入已有页面

3. 选中你之前导出的 .html 文件

4. 所有内容、图片、配色恢复成可编辑状态，接着改即可

常见问题

Q：双击没反应 / 显示乱码？
右键文件 →「打开方式」→ 选 Chrome 或 Edge。
Q：导出图片是空白 / 报错？
确认电脑联网；如果图片是用网址引用的，改成「📷 上传本地图片」即可。
Q：能在手机上用吗？
可以查看，但编辑建议在电脑上操作，体验更好。
