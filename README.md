# Onepager Editor · 个人/公司 主页编辑器

一个**单文件网页编辑器**，用来制作和修改个人 / 公司主页卡片（一页式 one-pager）。可导出干净网页或 3 倍高清图片，用于朋友圈、小红书、微信群、官网等。

无需安装、无需懂代码 —— 打开网页直接「点文字改文字、悬停换图片、拖积木排版」。

## 在线使用

- **编辑器**：https://lesleyfung.github.io/onepager-editor/
- **使用说明**：https://lesleyfung.github.io/onepager-editor/%E4%BD%BF%E7%94%A8%E8%AF%B4%E6%98%8E.html

## 功能

- **积木式排版** —— 头部、数据、客户墙、合作步骤、联系卡片等模块可上移 / 下移 / 复制 / 删除
- **点改文字、悬停换图** —— 所有文字直接点开编辑；图片可缩放、切圆/方角、上传本地图
- **二维码** —— 联系卡片一键添加微信 / 飞书 / 公众号等二维码位
- **主题** —— 8 个预设主题色 + 自定义取色器，多种背景调与字体
- **导出** —— 「导出网页」生成不带编辑按钮的干净 `.html`；「导出高清图片」生成 3 倍 PNG
- **续编** —— 「导入已有页面」可把之前导出的成品重新载入继续编辑

> 编辑与「导出网页」完全离线可用；「导出高清图片」需联网（使用 html2canvas CDN）。

## 本地使用

下载 `index.html` 双击用浏览器打开即可（推荐 Chrome / Edge）。详见 [使用说明](使用说明.md)。

## 部署

纯静态站点，已部署在 GitHub Pages（`main` 分支根目录）。Fork 后在仓库 **Settings → Pages** 选择 `main` 分支即可获得自己的在线地址。
